# Minimal Reqs for Product

Ziel ist es, eine echte Alternative zu Goodreads zu entwickeln.
Aktuell leidet Goodreads unter einer unübersichtlichen Oberfläche, einem unvollständigen Buchindex – und die Performance ist schlicht katastrophal.

Für ein minimales Produkt wird zunächst ein Nutzer-System benötigt.
Ein Nutzer soll in der Lage sein, seinen Lesestatus zu verfolgen: gelesen, ungelesen oder aktuell am Lesen.

Als zweites großes Feature kommen grundlegende Social-Media-Funktionen hinzu.
Aktivitäten wie Lesefortschritte oder Statusupdates sollen mit anderen Nutzerinnen und Nutzern geteilt werden können.

- User-System
  - Registrierung und Login
  - Nutzerprofil
  - Lesestatus für Bücher:
    - Gelesen
    - Ungelesen
    - Aktuell am Lesen

- Social Features (Minimal)
  - Teilen von Lese- und Statusupdates
  - Sichtbarkeit für andere Nutzer*innen
